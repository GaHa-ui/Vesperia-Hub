# Vesperia Hub

Fabric mod for Minecraft 1.21.1+ with PvP visual effects.

## Features

- **Hit Particles** - Custom particles on hit
- **Target HUD** - Display target HP, armor, distance
- **Damage Numbers** - Floating damage text
- **Critical Effects** - Visual feedback on crits
- **Trajectory Prediction** - Arrow trajectory line

## Requirements

- Minecraft 1.21.1+
- Fabric Loader 0.16.0+
- Java 21

## Installation

1. Build with `./gradlew build`
2. Copy JAR from `build/libs/` to your mods folder

## Building

```bash
./gradlew build
```

JAR will be in `build/libs/vesperia-hub-1.0.0.jar`
